\# Power BI Module Project – The Billion Dollar Club



\## Overview

This project is an interactive Power BI dashboard analyzing global unicorn companies (privately held startups valued at $1 billion or more), completed as part of the Torilo Academy Data Analysis Program. It demonstrates end-to-end Power BI skills — from data cleaning to DAX measures to interactive visual design.



\## Objective

The goal was to explore the "Billion Dollar Club" of unicorn companies and build a dashboard that surfaces key insights: which industries produce the most unicorns, which countries lead in unicorn creation, valuation trends over time, and how quickly companies reach unicorn status.



\## Tools Used

\- \*\*Power BI Desktop\*\* – dashboard design and data visualization

\- \*\*Power Query\*\* – data cleaning and transformation

\- \*\*DAX (Data Analysis Expressions)\*\* – custom KPI measures





\## Dataset

The dataset used is `Unicorn\_Companies\_Raw\_Dataset.csv`, located in the `Data/` folder. It contains company-level data including valuation, industry, country, and founding/unicorn dates.

\## Dashboard Features

\- \*\*4 KPI measures\*\* built with DAX and organized in a dedicated `\_Measures` table for clean, maintainable calculations

\- \*\*4 core visuals\*\* covering valuation trends, industry breakdown, and geographic distribution of unicorns

\- \*\*2 interactive slicers\*\* allowing users to filter the dashboard by criteria such as industry or country

\- Custom handling for map visuals and log-scale scatter charts to properly display valuation outliers



\## How to Use This Project

1\. Open `PowerBI\_Project\_File.pbix` in Power BI Desktop.

2\. Explore the dashboard using the slicers to filter by industry, country, or valuation range.

3\. Refer to `Analysis\_Report.pdf` for the full breakdown of methodology, DAX formulas, and key findings.



\## Key Insights

The dashboard highlights which industries and countries dominate unicorn creation, how valuations are distributed across the global unicorn landscape, and patterns in how quickly companies reach billion-dollar status. Full details are available in the Analysis Report.
